10 ## Description
11 
12 A subproject of Machine Intelligence Core, being the real "core" of the framework.
13 
14 Contains tools required for development of MIC-based applications, such as configuration management, properties, loggers, application state, event handlers etc.
15 
16 ### Main modules
17 
18  * application - classes related for management of applications, their state, key-handlers as well application factories.
19  * configuration - classes responsible for configuration management (parameters server, property tree etc.)
20  * logger - classes and functions related to logger
21 
22 ### Applications
23 
24  * logger_test - application for testing console logger.
25

51 ## Installation of MI-Toolchain
52 The following assumes that all MIC modules will be installed to the directory '../mic'.
53 
54  git clone git@github.com:IBM/mi-toolchain.git
55  cd mi-toolchain
56  ./scripts/build_mic_module.sh ../mic
57 
58 ### Make commands
59 
60  * make install - install applications to ../mic/bin, headers to ../mic/include, libraries to ../mic/lib, cmake files to ../mic/share
